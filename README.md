# Detective Daily

Detective Daily is an AI-powered daily detective game for Reddit. Each day, a new mystery is generated, and players must investigate the scene, interrogate suspects, and gather clues to solve the case.

## Features
- Daily procedurally generated mysteries
- Interactive interrogations using LLM integration
- Global leaderboards and daily streaks
- Push notifications for new content

## How to play
1. Install the app on your subreddit.
2. A new daily case will be posted automatically.
3. Users can interact with the post to play the game!

## Fetch Domains
No external fetch domains are currently requested.
